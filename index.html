Overview of the internet

The internet consists of 4 major components
  1. Browsers:
    • Browser is a program that shows files on the Web
  2. The internet:
    • The internet is the largest computer network
  3. Servers:
    • Server is a program that responces a request from a browser and send files to the browser
  4. HTTP:
    • HTTP is a protocol used to talk between browsers and servers

Basic of the Web

Web pages are documents writtend in HTML. Browsers read and display the documents

  • HTML:
    • HTML is a markup language including tag, element and attribute. Most tags have an opening and closing tag but a few tags do not (called void tag, e.g. img, br..etc)
  • Element & Attiribute:
    • An element is contents within a tag. An attribute is a property of a HTML element
